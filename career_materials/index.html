研究项目 智能电动汽车 自适应界面

面向里程焦虑的智能电动汽车自适应界面与驾驶员压力画像研究

针对智能电动汽车里程焦虑引发的认知压力，提出一种基于客观行为与生理特征的自适应交互框架，探索由驾驶员压力画像驱动的动态干预策略。

  • 自主设计模拟驾驶高压力诱发实验，围绕低电量与信息负荷构建可控压力场景。
  • 采集多维眼动、行为序列与驾驶表现数据，用生理数据画像刻画驾驶员压力差异。
  • 确立“认知画像识别-驾驶员聚类-自适应干预映射”的研究路线。

校企合作项目 情感对话评估与验证

车载情感陪伴智能体评分体系

围绕车载语音情感陪伴场景，构建一套可解释的评分框架，并通过多轮Bench样本和LLM-as-a-Judge流程验证高低质量回复的区分能力。

  • 搭建包含安全门控、单轮质量、多轮轨迹质量和用户状态扭转的完整评分维度。

  • 设计LLM-as-a-Judge输入输出结构，要求模型给出维度分、理由说明、错误标签和最终综合分。

AI产品 个人Vibe Coding

Echo-AI社交焦虑认知训练平台

基于CBT认知行为疗法框架，Echo把真实社交困扰转化为可复盘、可模拟、可追踪的AI训练流程，帮助用户识别消极思维模式，并在低风险环境中练习社交表达。

情景复盘

用苏格拉底式提问引导用户拆解情绪、自动想法、证据与替代解释，把“反复想”转化为可操作的认知重构流程。

社交模拟

提供寝室、课堂、社团、面试等低风险训练场景，并在用户卡住时支持即时求助和回复建议。

成长记录

沉淀训练次数、情绪觉察、认知偏差类型和阶段性状态，让用户看到持续练习带来的变化。

分析报告

结束对话后生成0-100分综合评分、认知偏差总结、亮点进步和下一步行动建议。

愿景
提供低风险、可反复练习的AI认知训练环境。
目标人群
18-25岁大学生，尤其适合正在适应新社交环境、对反馈敏感，或面临面试、小组合作、课堂表达等高压力沟通任务的人。
技术栈
React 19、Vite 8、CSS、React-Markdown、FastAPI、SQLAlchemy、SQLite、LLM API。

PooLOG

关注肠道健康，从每一次记录开始。

PooLOG是一款专注肠道健康管理的个人数字伙伴，通过饮食、饮水、排便、经期和心情记录，帮助用户把零散的身体信号沉淀为可回顾、可理解、可行动的健康线索。

交互设计课程小组作业 桌面调研 / 问卷 / 访谈 手绘线框图 - 低保真 - 高保真
01 需求发现

通过桌面调研、问卷调查和访谈梳理肠道健康记录的真实痛点，明确用户对低负担记录、异常提醒和长期趋势回顾的需求。

02 原型迭代

从手绘线框图到低保真，再到高保真界面，围绕记录效率、信息层级、入口可见性和情绪化表达进行了三轮迭代。

03 体验表达

用轻量化表单、日历回顾、附近厕所地图、健康报告和数字伙伴反馈，降低健康记录的尴尬感与坚持成本。
